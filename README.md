# Grapevine J
Grapevine J is an early-into-development Java 21 port of Grapevine 3, 
which was developed by Adam Cerling in Visual Basic 6. 
In its current state, Grapevine J is a command-line application; the development of
an appropriate front-end interface is on the roadmap. See ROADMAP.md.
